Git y GitHub

Concepto de Git

Git es un sistema de control de versiones distribuidas. Esto significa que cada desarrollador tiene una copia del historial del proyecto en su equipo local y no depende de un servidor para trabajar.

Concepto de GitHub

GitHub es una plataforma en la web que permite almacenar y gestionar proyectos que utilizan Git como sistema de control de versiones. Fue lanzada en el 2008 y adquirida por Microsoft en en 2018. Es la plataforma más popular en el mundo para alejar código, colaborar en proyectos de softward y gestionar versiones de manera remota.
